時程 規劃

從介面繪圖基礎開始，到網頁切版動畫，在回到後台資料的框架

  • UI/UX
  • 切版動畫
  • 前端後端
  • 第一週
    illustrator   icon

    先拿起武器

    介紹軟體工具的使用，業界常使用的工具illustrator與sketch的差異，做出屬於你自己的一套icon吧

  • 第二週
    創意發想，一顆愉悅的心

    挖掘內心，等待靈感的來臨

    解放自己的想像，發散具體的事物收斂成具體的感受
  • 第三週
    UI,html基本概念

    動手吧！

    了解基本的網站限制，再畫出相對的介面
  • 第四週
    UI guile

    在拓展思考維度

    觀察市場，了解趨勢的介面，訂定框架
  • 第四週
    html, css

    基礎的語法帶你做出第一個網站

    以Airbnb為例，做出相對應的元件
  • 第五週
    html, css

    漸漸成形，一個完整的商業網站

    遵照文件，有脈絡的完成你的設計
  • 第六週
    html, css ,jQuery

    加點動態元素，讓你的網站能夠與使用者互動

    使用css偽元素來新增動畫，在用jQuery的套件完成動畫效果
  • 第七週
    Responsive Web design

    有了網站，你準備好接受不同的裝置和使用情境嗎？

    針對不同行動裝置，探討使用者的邏輯與習慣，資訊架構的延伸。
  • 第八週
    jQuery

    只有空殼是不夠的，讓介面有了功能

    動態的資料轉換，了解js的基礎訓用
  • 第九週
    UI,html,css,jQuery,bootstrap

    回顧所有的學習，迅速的打造你的作品集or商業網站

    學習bootstrap和常用plugin，敏捷的打造一個網站
  • 第十週
    SEO,Adword,金流

    網站有了雛形，上線測試市場吧

    上線前該注意什麼，怎麼讓人找到你，自傳要曝光，商業網站需串金流
  • 第十一週
    python,django

    初次認識後台

    此次的教學以實戰一個內容行後台，不含相關基礎，已開啟後台邏輯為考量，若期望學習更多相關觀念，請點選網路課程共學方案
